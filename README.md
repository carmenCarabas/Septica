# Septica – Multiplayer Card Game (C#)

## Description
This project is a digital implementation of the Romanian card game "Septica".  
It includes both single-player and multiplayer modes, with real-time communication between clients.

## Features
- Single-player mode with game logic implemented locally
- Multiplayer mode using client-server architecture
- Real-time communication between players
- Object-Oriented Programming (OOP) design

## Project Structure
- /Main  
  Contains the core game logic, main classes, and single-player implementation.  
  Also handles the server-side logic for multiplayer.

- /Client
  Handles the client-side interface and communication with the server.

## Technologies Used
- C#
- .NET Framework
- TCP/IP (sockets)
- OOP principles

## How to Run
1. Start the server from the Main module
2. Launch client instances to connect to the server
3. Play in multiplayer or single-player mode
